# nexsys4_ddr board files (tutorial 3)

1.  copy `imports/board_files/nexsys4_ddr` to `<Vivado Install Directory>/Xilinx/Vivado/<Vivado Version>/data/boards/board_files`
2.  Alternatively, via TCL console: `set_param board.repoPaths <path to extracted board files>/board_files/` (this adds board files to current instance of Vivado)
